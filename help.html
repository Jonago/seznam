How to search

Each search field describes which element will be searched for. In the All field it is possible to search by all elements (words and numbers).

On smaller screens following search fields are omitted (Name, Last Name, Field of endeavour and Country). Instead, All field can be used to search omitted elements.

Search is not strict. The more characters is written, the more precise results it will render.

While searching with the POD field and when selected one decimal number, results will show also 2nd decimal number with it. For example: when selected 1.7 it will show also numbers 1.75 and 1.76. But when selected 1.75 only that result will be shown.

When an element in the head of the table is clicked (for example Name or POD) it will sort data by ascended or descended order. The arrow beside element indicate the order. By default the list is sorted by names in ascended order.

It is possible to search by multiple search requirements. For example: find all initiates who have 2nd soul ray and 5th mental ray.

Reset button resets all search fields. This is useful when search fields are overcrowded with data. Wait couple of seconds to reset the fields. Refreshing the browser gives the same result. It is also possible to reset individual field (except for the rays).

Number at right bottom side of the form shows all persons in the list, while number in the red circular badge represent number of persons we are searching for.
